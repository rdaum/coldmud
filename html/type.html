type

type(data)
This function returns a symbol giving the type of data. The return value is one of 'integer, 'string, 'dbref, 'list, 'symbol, or 'error.

Examples:

type(5)
     => 'integer
type([6, 'foo, "bar"])
     => 'list
type(<#44, #[['type, 'obj], ['obj, #20]]>)
     => 'frob
