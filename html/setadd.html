setadd

setadd(list, data)
This function returns the result of adding data to the end of list if it was not already somewhere in list. If data was already in list, then setadd() returns list unmodified.

Examples:

setadd([2, 3, 4], 'foo)
     => [2, 3, 4, 'foo]
setadd([2, 3, 4], 3)
     => [2, 3, 4]
